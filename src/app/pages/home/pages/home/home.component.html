About

SSB ISO Information Management System is an integrated system for managing ISO forms and procedures. It has data entry, management of documents between departments, reports generation and data analysis capabilities. Below is a summary description of each of available modules.

Dashboards

This is an important module where one lands and get the saved reports in form of charts and tables. One can also view the definition of the indicator used.

Documents

This is a repository for resources of types forms or procedures are managed via this module. You can download the forms or procedures from here and use them.

Data entry/Upload

SSB ISO IMS enables you to enter data by numbers but also upload the document you have worked with and you need to send it to the supervisor/manager or another department.

Reports

Through this module you can view shared standard reports and uploaded reports but also get access to the analysis tools which are Pivot table and Data visualizer. Pivot table enables you to generate reports in form of tables and Data visualizer gives you charts or graphs
